# 💎 QR Code Tracking for Gemstones

ကျောက်မျက်ရတနာ တစ်လုံးချင်းစီကို စနစ်တကျ ခြေရာခံနိုင်ဖို့အတွက် QR Code စနစ် အသုံးပြုပုံ လမ်းညွှန်ဖြစ်ပါတယ်။

## 🛠 Features

### 1. QR Generation (`gemstone_qr_widget.dart`)
- ကျောက်မျက် ID သို့မဟုတ် Unique QR String အပေါ် အခြေခံပြီး QR Code ထုတ်ပေးသည်။
- App Logo ကို QR ၏ အလယ်တွင် ထည့်သွင်းနိုင်ပြီး Professional ဆန်စေသည်။
- ကျောက်မျက်အသစ် ထည့်သွင်းချိန်တွင် အလိုအလျောက် QR Code ထုတ်ပေးမည်။

### 2. QR Scanning (`qr_scanner_impl.dart`)
- Mobile Camera ကို အသုံးပြု၍ QR Code များကို အမြန်နှုန်းဖြင့် စကင်ဖတ်နိုင်သည်။
- Flashlight ထိန်းချုပ်မှုနှင့် ရှေ့/နောက် ကင်မရာ ပြောင်းလဲမှုများ ပါဝင်သည်။
- စကင်ဖတ်ပြီးသည်နှင့် သက်ဆိုင်ရာ ကျောက်မျက်၏ အချက်အလက်များကို Database မှ ရှာဖွေပြသပေးမည်။

## 🚀 Workflow

၁။ **Label Printing**: ကျောက်မျက်တစ်ခုကို Inventory ထဲ ထည့်လိုက်သည်နှင့် QR Code ကို Print ထုတ်ပြီး ကျောက်ထည့်သည့် အိတ် သို့မဟုတ် ဘူးပေါ်တွင် ကပ်ပါ။
၂။ **Quick Search**: ကျောက်မျက်၏ အခြေအနေ (ဥပမာ - သွေးပြီး/မပြီး) ကို သိလိုပါက App ရှိ Scanner ဖြင့် စကင်ဖတ်လိုက်ရုံဖြင့် အချက်အလက်အားလုံး ပေါ်လာမည်။
၃။ **Processing Tracking**: ကျောက်မျက်ကို အလုပ်သမားထံ ပေးအပ်ချိန် သို့မဟုတ် ပြန်လည်လက်ခံချိန်တွင် QR စကင်ဖတ်၍ မှတ်တမ်းတင်ပါ။

## 🛠 Required Packages
ဤလုပ်ဆောင်ချက်အတွက် အောက်ပါ Package များ လိုအပ်ပါသည်-
- `qr_flutter`: QR Code များ ထုတ်ပေးရန်။
- `mobile_scanner`: QR Code များကို စကင်ဖတ်ရန်။
- `path_provider`: QR Image များကို သိမ်းဆည်းရန် (လိုအပ်ပါက)။
